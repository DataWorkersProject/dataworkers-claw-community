# Microsoft Copilot Setup — Community Edition

> Current status and compatibility notes for Microsoft Copilot (Copilot Studio).

## Current Status

**Status: Evolving.** Microsoft has announced MCP support for Copilot Studio. The integration is enterprise-oriented and aimed at organizations in the Microsoft 365 ecosystem.

## What We Know

- Microsoft announced MCP support for Copilot Studio in 2025.
- MCP support is expected to follow standard stdio transport, which Data Workers supports.
- Azure AI Foundry is the expected configuration platform.

## Compatibility

The Data Workers MCP server is **fully compatible** with any standard MCP implementation. When Microsoft's integration reaches GA, the same server binary will work.
